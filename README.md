Virtual CV — Marco Kruger Goncalves

A responsive, single-page portfolio website was built as an interactive virtual CV. This repository contains the full source code for the site, deployed live via GitHub Pages.

Live site: https://mgon12591.github.io/Virtual-CV/

About this project:
This project was developed as part of the Profile Development Assignment for the CMPG323 module. It presents some of the knowledge I have gathered throughout my studies into one.

Features this project offers:
- Direct buttons that takes the user to my GitHub Profile, my LinkedIn Profile as well as a PDF version of my CV.
- A responsive layout that can easily be view and navigated on Desktops,Laptops,Tablets and other mobile devices.
- An interactive navigation bar that can easily take you to the desired sections.
- Smooth Scroll animations that are smooth and aesthetically pleasing.
- A summary of the projects that I have worked on and a link to the repositories for easy access.
- A "Back-to-Top" button that can easily take the user back to the top if the want to.

Sections in this project include:
- The "About Me" section that describes me as a person and a summary of my CV.
- The "Education" section which shows all the education I have completed or still busy with.
- The "Skills" section which shows some skills that I feel I prefer that I have learnt throughout my studies.
- The "Experience" section that shows the business experience I have gained.
- The "Languages" section which showcases the languages I am fluent in and the lamguage I am learning.
- The "Contact Details" section that gives the user my contact details if the would like to contact me.
- The "References" section that shows two references that can be contacted to verify the education I have gained as well as the experience.

This Project also makes use of:
- HTML5, which provides the page structure and content of this website. 
- CSS3, ensures for all the styling and animations of this website.
- JavaScript, ensures the scroll interaction and highlights the smooth effortless navigation of this website on mobile devices.
